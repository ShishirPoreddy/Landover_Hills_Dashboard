Municipal Budget Assistant

Ask questions about municipal budgets and get direct answers with evidence

About

This is a demo of the Municipal Budget Assistant widget. The widget allows residents to ask questions about municipal budgets and receive direct answers with supporting evidence and citations.

Note: This demo requires the backend API to be running on http://localhost:8000. Make sure to start the FastAPI server before testing the widget.

Features

Direct Answers

Get exact dollar amounts and totals for budget questions

Evidence & Citations

See supporting documents and line items for every answer

Smart Filtering

Automatically filters by department, year, and line item

Year-over-Year

Compare budget changes between fiscal years

Try These Questions

  • "How much is allocated to road repairs?"
  • "What's the Police budget for 2024?"
  • "Show me Public Works spending"
  • "Total budget for Administration"
  • "How much did we spend on equipment?"
  • "What changed in Police between FY2024 and FY2025?"

How to Use

Click the "Ask the Budget" button in the bottom-right corner to open the chat widget. Type your question and press Enter or click the send button.

The assistant will:

  • Parse your question to understand what you're looking for
  • Query the budget database for relevant information
  • Return a direct answer with supporting evidence
  • Show you exactly which documents and line items support the answer
  • Follow you across pages - the conversation persists as you navigate!

Integration

To add this widget to your own website, include the following script:

<script src="http://localhost:8000/public/assistant.js"></script>

The widget will automatically initialize when the page loads. You can also customize it:

<script> // Custom initialization const assistant = new BudgetAssistant({ apiUrl: 'http://your-api-url.com' }); </script>

API Endpoints

The assistant uses several API endpoints:

  • POST /ask - Ask questions and get answers
  • GET /summaries - Get budget summaries by department/year
  • GET /findings - Get validation findings and anomalies
  • GET /departments - List all departments
  • GET /years - List all fiscal years
  • GET /stats - Get basic statistics
